Read unsigned analog temperature. 

Sample \hyperlink{class_l_m35}{L\+M35} analog pin multiple times to find two identical samples to reduce noise. A maximum number of samples can be configured with macro L\+M35\+\_\+\+M\+A\+X\+\_\+\+S\+A\+M\+P\+L\+ES. The last sampled temperature will be returned when no identical temperatures found.

Temperature range\+: 0.\+0 .. 110 degree Celsius\+: A negative temperature cannot be measured, because the A\+DC pin can only sample between positive 0.\+0 and 1.\+1 Volt. \begin{DoxyReturn}{Returns}
Divide temperature by 10 to get the temperature integer, temperature modulo 10 results in the fraction, for example\+: int16\+\_\+t temperature = 182 means 18.\+2 degree Celsius.
